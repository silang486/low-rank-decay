# Low-Rank Decay (LRD) — 低秩衰减

针对尺度不变 Transformer 的优化器端谱衰减方法。

正在进行中的工作——此仓库包含一篇进行中论文的代码和早期实验，当前结果均为初步结果。

## 动机

现代 Transformer 使用 RMSNorm 和 QK-Norm，这使得模型的一部分具有尺度不变性：将权重矩阵乘以一个正标量不会改变该层计算出的函数。在这种情况下，普通的权重衰减只是缩小 Frobenius 范数——这是一种纯径向的移动，实际上并不会简化函数。任务梯度位于球面上（切向），而权重衰减将参数拉向原点（径向），两者几乎互不交互。

那么，如果径向衰减对尺度不变层效果不大，那么衰减应该作用于什么呢？我们的答案是：奇异值谱。

## LRD 做了什么

在正常的优化器更新之后，LRD 额外施加一步：

    W ← W - η·λ·polar(W)

其中 polar(W) = UVᵀ，W = UΣVᵀ。这相当于从每个奇异值中减去一个常数（软阈值化），使得较小的奇异方向被推向零，矩阵逐渐趋向更低秩。我们用几次 Newton-Schulz 迭代来近似 polar(W)，因此训练循环中无需进行 SVD。

对比：
- AdamW：σᵢ → (1-ηλ)σᵢ  （按比例缩小所有奇异值，谱形状不变）
- LRD：  σᵢ → max(σᵢ-ηλ, 0)  （软阈值化，重塑谱分布）

## 早期观察（模加法，p=113）

- LRD 会在测试准确率上升**之前**就显著降低 Q/K 矩阵的稳定秩——谱坍缩似乎先于泛化发生。
- 与 L2/AdamW 相比，LRD 似乎扩大了 grokking 仍能发生的低数据范围。

以上是单个任务上的单次运行观察。在得出可靠结论之前，仍需进行多种子运行、学习率对照、更强的基线以及因果测试。请将其视为一个方向，而非确定结论。

## 定位

LRD 可能不适合作为始终开启的预训练优化器——过早坍缩秩会丢弃后续需要的方向。它更适合用于中段训练 / SFT / CoT-SFT，此时已有广泛的表示，目标是巩固任务结构并去除类似记忆化的高秩噪声。
